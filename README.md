# ekstra11 | EKSTRA11.PL
Piłkarski draft przeglądarkowy stworzony dla fanów Ekstraklasy.

## O projekcie
EKSTRA11 to symulator draftu, w którym:
- Budujesz jedenastkę marzeń z puli ponad 600 zawodników.
- Wybierasz taktykę i losujesz kluby.
- Symulujesz sezon 34 kolejek ligowych.

## Technologie
- Czysty HTML/CSS/JS (Zero frameworków, czysta zabawa).
- Hosting: Netlify.
